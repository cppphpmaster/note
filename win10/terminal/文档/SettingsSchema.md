<p>Some Terminal settings allow you to specify custom background images and icons. It is recommended that custom images and icons are stored in system-provided folders and are referred to using the correct <a href="https://docs.microsoft.com/en-us/windows/uwp/app-resources/uri-schemes" rel="nofollow">URI Schemes</a>. URI Schemes provide a way to reference files independent of their physical paths (which may change in the future).</p>
<p>The most useful URI schemes to remember when customizing background images and icons are:</p>
<table>
<thead>
<tr>
<th>URI Scheme</th>
<th>Corresponding Physical Path</th>
<th>Use / description</th>
</tr>
</thead>
<tbody>
<tr>
<td><code>ms-appdata:///Local/</code></td>
<td><code>%localappdata%\Packages\Microsoft.WindowsTerminal_8wekyb3d8bbwe\LocalState\</code></td>
<td>Per-machine files</td>
</tr>
<tr>
<td><code>ms-appdata:///Roaming/</code></td>
<td><code>%localappdata%\Packages\Microsoft.WindowsTerminal_8wekyb3d8bbwe\RoamingState\</code></td>
<td>Common files</td>
</tr>
</tbody>
</table>

<p>⚠ Note: Do not rely on file references using the <code>ms-appx</code> URI Scheme (i.e. icons). These files are considered an internal implementation detail and may change name/location or may be omitted in the future.</p>
</blockquote>

<p>Terminal displays icons for each of your profiles which Terminal generates for any built-in shells - PowerShell Core, PowerShell, and any installed Linux/WSL distros. Each profile refers to a stock icon via the <code>ms-appx</code> URI Scheme.</p>
<blockquote>
<p>⚠ Note: Do not rely on the files referenced by the <code>ms-appx</code> URI Scheme - they are considered an internal implementation detail and may change name/location or may be omitted in the future.</p>
</blockquote>
<p>You can refer to you own icons if you wish, e.g.:</p>
<div class="highlight highlight-source-json"><pre>    <span class="pl-s"><span class="pl-pds">"</span>icon<span class="pl-pds">"</span></span> : <span class="pl-s"><span class="pl-pds">"</span>C:<span class="pl-cce">\\</span>Users<span class="pl-cce">\\</span>richturn<span class="pl-cce">\\</span>OneDrive<span class="pl-cce">\\</span>WindowsTerminal<span class="pl-cce">\\</span>icon-ubuntu-32.png<span class="pl-pds">"</span></span>,</pre></div>

</blockquote>

<p>To do so, specify your preferred <code>backgroundImage</code>, position it using <code>backgroundImageAlignment</code>, set its opacity with <code>backgroundImageOpacity</code>, and/or specify how your image fill the available space using <code>backgroundImageStretchMode</code>.</p>

<div class="highlight highlight-source-json"><pre>    <span class="pl-s"><span class="pl-pds">"</span>backgroundImage<span class="pl-pds">"</span></span>: <span class="pl-s"><span class="pl-pds">"</span>C:<span class="pl-cce">\\</span>Users<span class="pl-cce">\\</span>richturn<span class="pl-cce">\\</span>OneDrive<span class="pl-cce">\\</span>WindowsTerminal<span class="pl-cce">\\</span>bg-ubuntu-256.png<span class="pl-pds">"</span></span>,
